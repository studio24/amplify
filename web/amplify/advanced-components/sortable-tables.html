Considerations

The table element must include the data-component="sortable-table" data attribute to allow each column to be sortable.

Each table header must include the data-type="..." data attribute, which specifies the type of data in each column, e.g. data-type="number", data-type="string" or data-type="date". Use data-type="no-sort" when sorting functionality should be disallowed.

Where a table cell is holding a date, it must have a data-date attribute holding the date converted into UNIX timestamp, e.g. data-date="191030400".
